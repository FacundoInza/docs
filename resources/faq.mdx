## Getting Started

<AccordionGroup>
  <Accordion title="How does the 7-day free trial work?">
    Your 7-day trial starts after your onboarding demo — no credit card required. There are no automatic charges at the end of the trial period. Once your 7 days are up, we'll work with you to choose the plan that fits your operation. No surprises, no commitments.
  </Accordion>

  <Accordion title="Do I need technical experience to use Silver5 AI?">
    No. Silver5 AI is designed for P2P operators, not developers. Guided onboarding walks you through connecting your Binance account, configuring your first Positioning Bot, and setting up your team. Most users are operational within a single session. If you get stuck, support is available on every plan.
  </Accordion>

  <Accordion title="Does Silver5 AI work with all Binance accounts?">
    Yes — any Binance account with P2P trading enabled can be connected to Silver5 AI via API key. You configure the API key in your Binance account settings, then paste it into Silver5. The number of accounts you can connect depends on your plan (1 account on Basic and Pro, up to unlimited on Enterprise).
  </Accordion>
</AccordionGroup>

## Security

<AccordionGroup>
  <Accordion title="Does Silver5 AI have access to my funds?">
    No. Silver5 AI never has custody of your crypto and never requests withdrawal permissions. The API key you connect is configured without withdrawal access, which means Silver5 can interact with your P2P operations but cannot move funds out of your Binance account under any circumstances.
  </Accordion>

  <Accordion title="Is it safe to give Silver5 my API key?">
    Yes, when configured correctly. Silver5 AI requires only read access and P2P-level permissions — no withdrawal permission, no transfer permission. We recommend following the API key setup guide to ensure only the necessary permissions are enabled. Your API key gives Silver5 the ability to read order data and interact with P2P features, nothing more.
  </Accordion>

  <Accordion title="Can operators see my Binance credentials?">
    No. Operators you invite to Silver5 access the platform through their own Silver5 login — they never see your Binance API key, username, or password. All operator actions go through Silver5's access model, where you control what each team member can see and do. Every action is logged individually.
  </Accordion>
</AccordionGroup>

## Features

<AccordionGroup>
  <Accordion title="Can I use Silver5 AI with multiple Binance accounts?">
    Yes. Multi-account support is available on Intermediate plans and above. The Basic and Pro plans support 1 Binance account. Intermediate supports 2, Advanced supports 3, and Enterprise supports unlimited accounts. Each account is connected via its own API key.
  </Accordion>

  <Accordion title="Does the AI Agent need to be configured before it goes live?">
    Yes — a brief setup is required before the AI Agent handles real conversations. You'll configure a persona (name, tone, and communication style), a knowledge base (your policies, FAQs, and instructions), and the KYC mode (Link or Manual). A simulator mode lets you test the agent's behavior before enabling it for live orders.
  </Accordion>

  <Accordion title="What languages does the AI Agent support?">
    The AI Agent is multilingual. You configure the language and tone in the agent settings to match your customer base. Most Silver5 AI customers operate in Spanish and Portuguese, but the agent can be configured for any language your operation requires.
  </Accordion>
</AccordionGroup>
